Sapphire Sentinel

Sapphire Sentinel is a Linux command-line tool designed to help technicians, administrators, and support professionals document operational work directly from the terminal.

Its purpose is to make session activity, project work, updates, and technical changes easier to record in a clear and repeatable way, supporting stronger workflows and better operational transparency.
