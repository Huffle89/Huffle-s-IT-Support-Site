Projects and Portfolio

This page collects open Linux projects and real world work done through Huffle's IT Services. It serves both as a portfolio for clients and a home for open resources built on different Linux flavors.

Over time, entries here can link out to repositories, scripts, documentation and, later, to a private dashboard view for active clients.

Open Linux Projects by Flavor

These projects are designed for specific Linux flavors or families. Each entry lists the target distro, what problem it solves and what tools or languages were used.

Bazzite and Fedora Family

Target flavors: Bazzite, Fedora and compatible spins

  • Example: Gaming profile setup and maintenance script
    Automates installation of core gaming tools, drivers and QoL utilities on Bazzite or Fedora, with sensible defaults for updates and rollback awareness.
  • Tech used: Bash scripting, dnf, flatpak, basic systemd integration

Debian and Ubuntu Family

Target flavors: Ubuntu, Linux Mint, Debian based systems

  • Example: Home workstation hardening preset
    A repeatable checklist and scripts that lock down a personal desktop, set up backups, and add basic monitoring with clear documentation for non technical users.
  • Tech used: Bash, ufw, unattended upgrades, rsync or restic

Arch and Rolling Based Systems

Target flavors: Arch, Manjaro and similar rolling releases

  • Example: Comfort coded base install profile
    A documented install flow that builds a stable daily driver on top of Arch or a derivative, with opinionated defaults that favor clarity and recovery.
  • Tech used: pacman, AUR helpers, shell scripts, documentation overlays

Client and Company Portfolio

These examples describe the kind of work done through Huffle's IT Services. Client names are not listed, only the type of client, the problem and the shape of the solution.

Secure Home Network for a Veteran Household

Client type: Veteran family, mixed device environment
Focus: Security, clarity and low maintenance

Audited an existing home network, replaced default router settings, set up a safer Wi Fi layout and separated smart devices from work and personal machines. Documented the setup in plain language so the household could maintain it without stress.

Skills applied: CompTIA Network+, Security+, CySA+ mindset for risk reduction

Gaming and Streaming Desktop Refresh

Client type: Neurodivergent gamer
Focus: Performance, predictability and reduced overwhelm

Cleaned and optimized a gaming system, configured launchers and streaming tools, removed unnecessary startup clutter and set up simple restore points so the system could be returned to a known good state after changes.

Skills applied: CompTIA A+, hardware diagnosis, OS tuning and UX awareness

Small Business Tech Starter Stack

Client type: Local small business
Focus: Reliable basics and growth room

Helped set up a domain, basic email, a simple web presence and a practical backup plan. Outlined future steps for when the business grows, such as shared storage, access controls and more advanced monitoring.

Skills applied: A+, Network+, Security+, small business planning
